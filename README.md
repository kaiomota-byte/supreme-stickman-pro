# Stickman Combat Evolved - 2D Advanced Engine

Um clone moderno e otimizado inspirado em Supreme Stickman Duelist, desenvolvido com arquitetura pura ES6 Modulada e Física baseada no ecossistema Matter.js.

## 🚀 Como Executar Localmente (No Chromebook ou PC)

Como o projeto usa Módulos JavaScript nativos (`type="module"`), o navegador precisa de um servidor local básico para rodar por questões de segurança (CORS).

### Método Rápido para Chromebook:
1. Instale a extensão do Chrome: **Web Server for Chrome**.
2. Abra a extensão, escolha a pasta do projeto e ligue o servidor.
3. Acesse o endereço `http://127.0.0.1:8887` fornecido por ele.

---

## 🛠️ Publicação e Deploy no GitHub Pages
1. Crie um repositório Público no GitHub.
2. Faça o upload de todas as pastas e arquivos exatamente na mesma ordem estrutural.
3. Vá em **Settings -> Pages**.
4. Mude a branch para `main` / `root` e clique em **Save**.
